Устаревшая версия SQL Server, не поддерживается, создаёт риски безопасности.

**Проблемы:**
- Не поддерживается Microsoft
- Риски безопасности
- Ограниченные возможности масштабирования
- Высокие затраты на поддержку

**План миграции:**
- Миграция в облачный DWH (0-6 месяцев)
- Миграция операционных данных в PostgreSQL (6-12 месяцев)

**Бизнес-сценарии**: Все (блокирует)
